## Notes Importantes

1. **Usernames corrigés:** Les orthographes exactes sont `calebjay` (pas calebjaay), `eqavox` (pas egavox), `Petrosilia` (pas Petrosilla).

2. **URLs directes:** Chaque lien pointe vers la page exacte du son (sound_id inclus). Format: `https://freesound.org/people/{auteur}/sounds/{sound_id}/`.

3. **Localisations non renseignées par les auteurs:** 5 pistes n'ont pas de geotag sur freesound. Pour celles-ci une localisation a été:
   - **déduite** avec un bon niveau de confiance depuis les autres sons/profil de l'auteur: Pluie + Tente (Petrosilia — Bornholm), Cigales (pablodavilla — Lampedusa), Voiture (leonelmail — Québec)
   - **inventée** de façon cohérente avec le son car aucune donnée n'est disponible sur l'auteur: Plage (eqavox — Cornouailles)

   Ces localisations sont marquées *(déduit)* ou *(inventé)* dans chaque fiche pour référence.

4. **Sons partagés:** Les pistes Oiseaux/Rivière partagent le même fichier source (calebjay, Taïwan). Les pistes Pluie/Tente partagent également le même fichier source (Petrosilia).

5. **Licences:** Toutes les sources utilisent des licences créatives libres (Creative Commons 0 ou Attribution 4.0). Pour les sons sous Attribution 4.0 (Oiseaux et Rivière de calebjay), l'attribution à l'auteur est requise dans les crédits de l'app.
